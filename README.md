# Документация по использованию Telegram-бота «EnglishCard»

## Описание проекта
Telegram-бот «EnglishCard» предназначен для изучения английских слов в игровой форме. Бот позволяет пользователям тренироваться на базовом наборе слов, а также формировать свой личный словарь.

## Технологический стек
- **Язык программирования:** Python 3.x
- **Библиотека для бота:** pyTelegramBotAPI (telebot)
- **База данных:** PostgreSQL
- **Драйвер БД:** psycopg2

## Структура проекта
- `main.py` — основной файл запуска бота и обработки сообщений.
- `db_utils.py` — вспомогательные функции для работы с базой данных PostgreSQL.
- `database.py` — скрипт для первичной настройки базы данных и загрузки начальных слов.
- `tables.sql` — SQL-скрипт для создания структуры таблиц.

## Установка и запуск

### 1. Подготовка базы данных
Убедитесь, что у вас установлен PostgreSQL. Создайте базу данных (по умолчанию в коде используется имя `postgres`).
В файлах `db_utils.py` и `database.py` при необходимости обновите параметры подключения в функции `get_connection()`:
- `host`
- `database`
- `user`
- `password`

### 2. Инициализация таблиц и данных
Перед первым запуском бота необходимо создать структуру таблиц и загрузить 10 базовых слов. Для этого запустите файл `database.py`

### 3. Запуск бота
Укажите ваш API токен в переменной `TOKEN` в файле `main.py`, затем запустите бота.

## Инструкция по работе с ботом

1. **Начало работы:** Введите команду `/start`. Бот поприветствует вас и предложит первое слово для перевода.
2. **Тренировка:**
   - Бот присылает слово на русском языке.
   - Вам предлагается 4 варианта перевода на кнопках.
   - При правильном ответе бот подтвердит успех и пришлет следующее слово.
   - При ошибке бот предложит попробовать еще раз.
3. **Добавление слов:** Нажмите кнопку **«Добавить слово ➕»**. Сначала введите русское значение, затем английский перевод. Слово будет добавлено в ваш персональный список.
4. **Удаление слов:** Нажмите кнопку **«Удалить слово»** и введите слово, которое хотите исключить из своего рациона.
5. **Пропуск слов:** Кнопка **«Дальше ⏭»** позволяет пропустить текущее слово и перейти к следующему.

## Особенности реализации
- **Персонализация:** Новые слова, добавленные одним пользователем, не видны другим.
- **Изоляция:** Неправильные варианты ответов подбираются только из тех слов, которые есть в словаре конкретного пользователя.
